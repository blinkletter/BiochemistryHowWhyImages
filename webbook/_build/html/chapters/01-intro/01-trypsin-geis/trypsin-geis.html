Irving Geis’ Trypsin#

In the book I presented an edited version of Irving Geis’ 1974 drawing of Trypsin, based on x-ray crystallographic data available at the time. As you can see, it was accurate with respect to modern data sets.

I edited the image using Affinity Photo (now available as Affinity Studio). The working file is available as images/geis-0022-trypsin_color.afphoto in the GitHub repository with this noteboook

The image was exported in png format and the version presented in the textbook is reproduced below.
